# 🦕 Imparando le divisioni con un dinosauro

## Benvenuto nel Minigioco Educativo!

Questo è il gioco completo e pronto all'uso per insegnare le divisioni ai bambini in modo divertente e visivo.

### 🚀 Come Avviare

Apri semplicemente il file **`index.html`** in un browser moderno (Chrome, Firefox, Safari, Edge).

Il gioco si caricherà automaticamente e potrai iniziare a giocare!

### 🎮 Come Si Gioca

1. **Scegli un livello** dalla schermata iniziale
2. **Leggi il tutorial** per capire come giocare
3. **Trascina gli oggetti** (cibo, uova, frutti) sui dinosauri
4. **Distribuisci equamente** - ogni dinosauro deve ricevere la stessa quantità
5. **Raccogli le stelline** per ogni livello completato!

### 📚 Cosa Impara Tuo Figlio

✅ **Concetto di divisione** - Come distribuire equamente  
✅ **Pensiero logico** - Risolvere problemi passo per passo  
✅ **Numeri e quantità** - Conteggio e confronto  
✅ **Introduzione al resto** - Cosa succede quando non si divide perfettamente  

### 🎨 Design

Il gioco presenta:
- 🦕 Un dinosauro protagonista simpatico e riconoscibile
- 👥 Un branco colorato di 4 dinosauri (verde, arancione, blu, rosa)
- 🌴 Ambientazione cartoon preistorica
- ⭐ Sistema di ricompense con stelline
- 🎉 Celebrazioni animate quando completi un livello

### 📊 Livelli

**Livello 1-3**: Divisioni semplici (imparare le basi)  
**Livello 4-6**: Divisioni più complesse  
**Livello 7-8**: Introduzione al concetto di resto  
**Livello 9-10**: Problemi guidati e sfide finali

### 🆘 Risoluzione Problemi

**Il gioco non si carica?**
- Assicurati di aprire il file `index.html` (non i file in assets/)
- Prova un browser diverso
- Pulisci la cache del browser

**Il drag-and-drop non funziona?**
- Accertati che JavaScript sia abilitato
- Prova su un browser diverso
- Su mobile, assicurati di toccare e trascinare

**Il gioco è lento?**
- Chiudi altre applicazioni
- Aggiorna il browser
- Pulisci la cache

### 📖 Per I Genitori

Questo gioco è stato progettato specificamente per bambini di 4-8 anni. È completamente sicuro - non ci sono annunci, non richiede dati personali, non ha acquisti in-app.

**Suggerimento**: Gioca insieme a tuo figlio per la prima volta! Aiutalo a capire le meccaniche e poi lascialo giocare da solo.

### 🎓 Valore Educativo

Con questo gioco, tuo figlio avrà una comprensione **con le mani** di cosa significhi dividere. Non è solo teoria - è pratica concreta con feedback visivo immediato.
